Written by Rory Barnes

./vplanet is a general purpose tidal integrator. It takes an input file consisting of options and initial conditions and simulates tidal evolution, along with other secondary parameters, forward and/or backward in time. This help describes the basics of the input file.

----- Command Line Options -----

-v, -verbose -- Maximum verbosity, i.e. display all warnings and updates.
-q, -quiet   -- No verbosity, i.e. nothing printed to device.
-h, -help    -- Display this message.

----- Input File -----

Comments/White Space: Any characters to the right of a # sign are treated as a comment and are ignored. All white space is ignored.
Options that take an array may span multiple lines if a $ is placed at the end of the line.
Options must be the first string on any line, and must be written exactly as listed below.
Arguments may have any format, and need only be unambiguous.
