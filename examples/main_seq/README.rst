To run this example
-------------------

.. code-block:: bash

    vplanet vpl.in
    vplot

---------------

.. figure:: main_seq.png
   :width: 300px
   :align: center

   Evolution of the luminosity, radius, period, and temperature of M, K, and G dwarfs as a function of time
   according to the Baraffe et al. (2015) stellar evolution tracks.
